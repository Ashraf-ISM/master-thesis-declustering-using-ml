# Results Directory

Use this folder for generated thesis outputs.

## Layout

- `figures/`: plots, maps, and visual outputs
- `tables/`: summary tables and comparison outputs
- `reports/`: text summaries and generated analytical reports
- `catalogs/`: declustered or labeled catalog outputs

Each results area is further split into `supervised/` and `unsupervised/`.

## Current Unsupervised Outputs

- `figures/unsupervised/som_dbscan/`: SOM + DBSCAN plots
- `figures/unsupervised/gmt/`: GMT-derived map and visualization outputs
- `reports/unsupervised/som_dbscan/`: generated declustering report text
- `catalogs/unsupervised/som_dbscan/`: declustered output catalog
