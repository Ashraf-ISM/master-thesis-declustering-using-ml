# Data Directory

Use this folder for thesis datasets only.

## Layout

- `supervised/raw/`: original datasets for supervised workflows
- `supervised/processed/`: cleaned or feature-engineered supervised datasets
- `unsupervised/raw/`: original datasets for unsupervised workflows
- `unsupervised/processed/`: cleaned or feature-engineered unsupervised datasets

## Current Unsupervised Data

- `unsupervised/processed/som_dbscan/`: processed inputs and intermediate arrays for the SOM + DBSCAN workflow
- `unsupervised/processed/gmt/`: processed text files used for GMT-based visualization
- `unsupervised/raw/`: current raw GMT source files

Keep a short note with each important dataset covering source, date, region, and preprocessing assumptions.
