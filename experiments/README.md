# Experiments Directory

This folder stores exploratory work, comparison studies, and method-specific analyses that are not yet part of the final reusable pipeline.

## Layout

- `supervised/`: supervised experiments and comparison studies
- `unsupervised/`: unsupervised experiments and legacy exploratory work

## Current Unsupervised Contents

- `unsupervised/DBSCAN/`: DBSCAN notebook experiments
- `unsupervised/KDM/`: KDM notebooks, notes, and implementation file
- `unsupervised/HDBSCAN.ipynb`: HDBSCAN workflow exploration
- `unsupervised/NZ_Method*.ipynb`: New Zealand comparison notebooks
- `unsupervised/NZ_DeepLearning_Autoencoder.ipynb`: autoencoder-based unsupervised workflow
